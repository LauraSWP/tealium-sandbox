Adding Your Own Functions

Quick Steps:
  1. 1
    Open custom-functions.js in your editor
  2. 2
    Add your function using the template provided
  3. 3
    Register it in window.customFunctionRegistry
  4. 4
    Refresh the page to see your function in the UI
Function Template:
function agent_yourFunctionName() {
    // Your code here
    // Access to: utag_data, utag, logEvent, showToast
    
    utag_data.your_field = 'your_value';
    
    if (typeof utag !== 'undefined' && utag.view) {
        utag.view(utag_data);
    }
    
    showToast('Function executed', 'success');
}

// Register in customFunctionRegistry:
'Your Function Name': {
    func: agent_yourFunctionName,
    description: 'What your function does',
    category: 'Your Category'
}
